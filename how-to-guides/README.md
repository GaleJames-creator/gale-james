# How-to guides

## What how-to guides are:

How-to guides are simple, step-by-step instructions for doing a specific task. 

## When to use how-to guides:

Use how-to guides when you want to accomplish a specific, goal-oriented task by following clear, step-by-step instructions.

## Available how-to guides:

* [Implement retry logic for failed payments](./payment-retry-logic-.md): This guide provides intructions for setting up automatic retries for failed payments.
* [Process a refund](./payment-process-refunds.md): This guide shows you how to process a refund.
* [Manage security for webhook data transmission](./webhooks-data-transmission.md): This guide provides intructions for protecting webhook data.
* [Manage webhooks using the Admin UI](./how-to-guides/webhooks-Admin-UI.md): This guide explains how to created, edit, enable, disable, or delete webhooks, and manage secrets. 
* [Retrieve an anonymous shopper's order and product information](./webhooks-anonymous-shopper.md): This guide provides step-by-step instructions for retrieving an anonymous shopper's orders and product information.
