| Type | Purpose | User Goal |
|------|---------|-----------|
| üìö **[Tutorials](./tutorials/README.md)** | Learning-oriented | "Teach me the basics" |
| üõ†Ô∏è **[How-to guides](./how-to-guides/README.md)** | Task-oriented | "Help me solve this problem" |
