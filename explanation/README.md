# Explanation

This section contains explanatory content. See the [portfolio README](../README.md) for an overview of the Diátaxis framework used to organize this portfolio.

## What explanations are

Explanations focus on providing background, context, and conceptual understanding of a subject. It answers the question “why?” rather than “how?”

## When to use explanations

Use an explanation when you need to understand why things work the way they do, and want deeper background knowledge to make sense of the how-to guides, tutorials, or reference materials.

* [`Idempotency-Key`](../reference/Idempotency-Key): Covers the `Idempotency-Key` header format, requirements, expiration window, duplicate key behavior, and idempotency error codes.
* [Understanding webhooks](./understanding-webhooks.md): Covers the key webhook concepts, how they work, and when to use them.
